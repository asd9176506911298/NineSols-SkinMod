# Nine Sols Skin Mod

# How to Use
- Press F1 select you want skin

# Skin
- DanceYi
- JieChuan
- Heng
- Jee
- Chiikawa Usagi
- Chiikawa Goblin
- AttackEffect
- Custom picture or GIF

# Shortcut
- Enable Skin (Shift + Q) 
- Create Custom Pic or Gif(Ctrl + Q) 
- you can change Shortcut in configEditor or using BepinExConfigurationManager
